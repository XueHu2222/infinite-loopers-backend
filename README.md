# Infinite Loopers Backend


## Commands

**Run from ROOT(/infite-loopers-backend):**
```bash
npm run setup        # First time: install + create databases
npm run dev          # Start all 5 services
npm run dev:gateway  # Start gateway only
npm run dev:users    # Start users only
npm run dev:tasks    # Start tasks only
npm run dev:achievements  # Start achievements only
npm run dev:shop     # Start shop only
```

## Ports

- Gateway: 3011
- Users: 3012
- Tasks: 3010
- Achievements: 3020
- Shop: 3014
